# Installation

This page covers client installation. For infrastructure deployment, use [Self-Hosting Guide](/self-hosting).

## Desktop App

Download latest desktop builds from GitHub Releases:

- [Latest Release Assets](https://github.com/zentra-chat/zentra-desktop/releases/latest)

Supported packages:

- **Linux:** AppImage / deb
- **macOS:** dmg
- **Windows:** msi / exe

### Linux (AUR)

If you use Arch-based Linux, you can install from AUR:

```bash
yay -S zentra-desktop-bin
```

## Web App

Run Zentra directly in the browser from your hosted frontend URL.

If you are hosting your own frontend/backend, continue with [Self-Hosting Guide](/self-hosting).
